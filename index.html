Misiones

Sistema de Gestión de Artículos

Este proyecto fue desarrollado en equipo para Programación 3 como una aplicación para la gestión de artículos. Permite agregar, eliminar y modificar artículos, así como cargar múltiples imágenes para cada uno. Los artículos se listan y se puede ver el detalle de cada uno. Además, incluye funciones para agregar y eliminar marcas y categorías a los artículos.

Para su desarrollo, utilizamos .NET Framework, C#, y SQL Server, y colaboramos a través de GitHub.
